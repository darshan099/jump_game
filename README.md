# jump_game
a simple game using graphics in C++

File name: jump.cpp

Summary: control the ball to land on the running blocks. If you miss a block, the game ends. Get the stats of the game at the end.

Controls: 
"a": left
"w": up
"d": right
"s": down

additional header files used: 
1) graphics.h
2) process.h
3) time.h
4) stdlib.h
